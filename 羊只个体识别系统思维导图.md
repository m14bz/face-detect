# 羊只个体识别系统

## 核心目标
- 训练后得到羊的512或1024维权重文件
- 新羊加入时，通过面部、侧脸、侧身、俯视等多角度的7-10张照片获取向量数据
- 在摄像头输入视频流中对每只羊进行个体判断
- 对比数据确定是哪只羊

## 技术架构

### YOLOv8 目标检测
- 模型选择：nano/small/medium/large/xlarge
- 现场部署推荐：YOLOv8n (CPU) 或 YOLOv8s (GPU)
- 性能：RTX 3090上YOLOv8n可达200+ FPS

### Re-ID 重识别系统
- 骨干网络：ResNet50（预训练ImageNet）
- 特征维度：512维或1024维可选
- 损失函数：ArcFace Loss（角度间隔损失）
- 特征归一化：L2归一化

## 完整工作流程

### 1. 数据准备
- 收集羊只多角度图像（7-10张/只）
- 存放于 `cropped/` 目录
- 运行 `scripts/prepare_reid_data.py` 整理数据
- 按8:2分割训练集和验证集

### 2. 模型训练
- 运行 `train_reid.py` 训练Re-ID模型
- 参数：`--feat-dim 512` 或 `--feat-dim 1024`
- 输出：`best.pt` 和 `last.pt` 权重文件
- 保存目录：`outputs/reid/`

### 3. 特征库构建
- 运行 `reid_inference.py build`
- 提取每只羊的平均特征向量
- 保存到 `gallery.pt` 特征库文件

### 4. 实时识别
- 运行 `reid_video_yolo.py`
- 集成YOLOv8检测与Re-ID识别
- 实时显示羊只ID和相似度

## 关键文件

### 训练相关
- `train_reid.py`: Re-ID模型训练脚本
- `scripts/prepare_reid_data.py`: 数据准备工具

### 推理相关
- `reid_inference.py`: 特征库构建与单张识别
- `reid_video_yolo.py`: 视频流实时识别

### 数据相关
- `cropped/`: 羊只图像存储目录
- `reid_dataset/`: 整理后的训练数据
- `outputs/reid/`: 模型和特征库存储

## 添加新羊只流程

### 1. 收集图像
- 为新羊只收集7-10张多角度图像
- 存放于 `cropped/G00100/` 目录

### 2. 重新准备数据
```bash
python scripts/prepare_reid_data.py
```

### 3. 重新训练模型
```bash
python train_reid.py --epochs 60 --feat-dim 512
```

### 4. 重新构建特征库
```bash
python reid_inference.py build --ckpt outputs/reid/best.pt --gallery-dir reid_dataset/train --save outputs/reid/gallery.pt
```

## 性能优化

### 特征维度选择
- **512维**：平衡精度和速度，推荐使用
- **1024维**：更高精度，但速度稍慢

### 硬件加速
- 使用GPU训练和推理（速度提升10-20倍）
- 确保安装GPU版PyTorch

### 识别参数调优
- 相似度阈值：默认0.5，可根据效果调整
- YOLOv8置信度阈值：默认0.25
- IOU阈值：默认0.45

## 常见问题

### 训练数据不足
- 增加数据增强
- 使用预训练模型微调
- 收集更多图像（10+张/只）

### 识别准确率低
- 增加训练轮数
- 调整相似度阈值
- 收集更多角度图像
- 使用更大特征维度（1024）

### 实时识别速度慢
- 使用GPU加速
- 减小特征维度
- 降低检测分辨率
- 使用更小的YOLOv8模型

## 技术细节

### Re-ID模型架构
- ResNet50骨干网络
- 2048维 → BatchNorm → 线性投影 → 512/1024维
- L2归一化输出

### 识别算法
1. 提取查询图像特征
2. 计算余弦相似度
3. 返回Top-K最相似羊只
4. 相似度≥阈值则识别成功

### 数据增强
- 随机水平翻转
- 颜色抖动（亮度、对比度、饱和度）
- 随机裁剪和缩放
